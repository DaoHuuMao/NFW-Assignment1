# 🎧 NFW Audio 

## 👤 Author
Group10

## 📁 Project Structure
This project involves multiple stages of audio signal processing using MATLAB and Python, including frequency analysis, compression, and jazz music generation.

### 🧩 Files Description

| File Name                | Description |
|-------------------------|-------------|
| `Part1_Recording.m`      | Record and save an audio signal. |
| `Part2_ShowSpectrum.m`   | Visualize the spectrum of the recorded audio. |
| `Part3_Compress.m`       | Apply Non-linear Frequency Warping and quantization to compress the audio. |
| `Part3_Decompress.m`     | Reconstruct the original audio from the compressed version. |
| `Part4_PNSRstatic.m`     | Calculate and evaluate PSNR for compression quality comparison (NFW vs MP3). |
| `Part5_JazzMixed.m`      | Mix jazz-style audio and generate final output. |
| `generate_jazz_midi.py`  | Python script to generate random jazz-style MIDI music. |

## 📌 Technologies
- **MATLAB** for signal processing tasks.
- **Python (mido library)** for MIDI generation.

## 🎯 Purpose
- Explore different audio processing techniques.
- Apply compression using Non-linear Frequency Warping.
- Generate jazz-style music automatically.
- Compare compression methods using PSNR.

## 📝 How to Run
1. Open MATLAB and run scripts in order: `Part1_Recording.m` → `Part2_ShowSpectrum.m` → `Part3_Compress.m`, etc.
2. Run `generate_jazz_midi.py` using Python (ensure `mido` is installed). 
3. Using timidity to convert midi to wav
4. Listen and compare output results.
